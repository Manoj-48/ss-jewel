# Supabase Integration Guide: User Tracking for SS Jewel Manufactures

To track how many people visit your catalogue or which items they click on, Supabase is the best free and private tool. Follow these steps:

## 1. Create a Supabase Project
1. Go to [Supabase.com](https://supabase.com) and sign in.
2. Click **New Project** and name it "SS Jewel Tracking".
3. Once created, go to **Project Settings** > **API**.
4. Copy your `Project URL` and `anon public` API Key.

## 2. Add the Supabase Script to `index.html`
Paste this at the top of your `<script>` section in your `index.html`:

```javascript
// Add Supabase Library
const script = document.createElement('script');
script.src = 'https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2';
document.head.appendChild(script);

let supabase;
script.onload = () => {
  // Replace with YOUR actual keys from Supabase
  const SUPABASE_URL = 'YOUR_SUPABASE_URL';
  const SUPABASE_KEY = 'YOUR_SUPABASE_ANON_KEY';
  supabase = supabase.createClient(SUPABASE_URL, SUPABASE_KEY);

  // Track Page View automatically
  trackEvent('page_view', { page: 'home' });
};

// Tracking Function
async function trackEvent(eventName, eventData) {
  if (!supabase) return;
  const { error } = await supabase
    .from('user_tracking')
    .insert([{ event_name: eventName, data: eventData, timestamp: new Date() }]);
  if (error) console.error('Tracking Error:', error);
}
```

## 3. Set Up the Database Table
In your Supabase Dashboard:
1. Go to **SQL Editor**.
2. Paste and run this command:
```sql
create table user_tracking (
  id bigint generated by default as identity primary key,
  event_name text,
  data jsonb,
  timestamp with time zone default timezone('utc'::text, now())
);

-- Turn off Row Level Security (RLS) for testing, or set to 'public insert'
alter table user_tracking enable row level security;
create policy "Allow public inserts" on user_tracking for insert with check (true);
```

## 4. Track Specific Clicks (Optional)
To track when someone clicks "Order on WhatsApp", update your `openModal` or `orderWA` function:
```javascript
function orderWA(id) {
  const p = PRODUCTS.find(x => x.id === id);
  trackEvent('whatsapp_click', { model: p.model, name: p.name });
  // ... rest of your code ...
}
```

## 5. Security Note
Never share your **Service Role Key**. Only use the **Anon Public Key** in your HTML file.
